# 🏦 Banking Simulation System

A **Banking Simulation System** built using **Python, Tkinter, and SQLite** that simulates basic banking operations like account creation, deposits, withdrawals, and balance checking.

This project demonstrates how real-world banking systems manage accounts and transactions while using a graphical interface and a database for data storage.

---

# 📌 Project Overview

The Banking Simulation System allows users to perform essential banking operations through a simple graphical interface.
It stores account details securely using an SQLite database.

This project helps in understanding:

* GUI development with Tkinter
* Database management with SQLite
* Real-world banking logic
* Python application structure

---

# ⚙️ Features

✔ Create new bank account
✔ Secure login with password
✔ Deposit money
✔ Withdraw money
✔ Check account balance
✔ Store data in SQLite database
✔ Simple GUI interface using Tkinter
